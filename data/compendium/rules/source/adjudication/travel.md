# Travel

Adventures often involve long-distance travel lasting hours or days. The GM can summarize such travel narratively rather than calculating exact distance and time in detail.

When exact second-by-second motion matters, use combat movement rules instead.

Travel uses the party's marching order and can involve:

- overland movement
- dungeon or wilderness transition
- vehicle-based travel

The `Travel Pace` subsystem is currently deferred and tracked in `docs/excluded_rules.md`, so this file records only the general travel framework for now.
