# Advantage and Disadvantage

When a roll has Advantage, roll two d20s and use the higher. When a roll has Disadvantage, roll two d20s and use the lower.

## Common Uses in the First Cut

- Hidden or favorable positioning
- Harmful conditions
- Species traits such as Brave or Gnomish Cunning
- Weapon properties or class features

## Non-Stacking Practical Rule

Even if multiple sources grant Advantage, the roll still uses only one extra d20. The same is true for Disadvantage. Conflicting Advantage and Disadvantage generally cancel each other before the roll is made.

## Prototype Guidance

Because many first-cut scenarios rely on stealth, charm resistance, and weapon tactics, Advantage and Disadvantage should be treated as first-class structured state during adjudication rather than as vague narration.
