# Interacting with Objects

Many object interactions are simple: the player describes the action, and the GM describes the result.

An object for rules purposes is a discrete inanimate item such as:

- a window
- a door
- a sword
- a book
- a table
- a chair
- a stone

Buildings and vehicles are not single objects for these rules because they are made from many objects.

Time-limited object interactions:

- when time is short, such as in combat, a creature gets one free object interaction per turn
- that interaction must occur during the creature's movement or action
- additional interactions require the Utilize action

Finding hidden objects:

- searching for a hidden door, trap, or similar feature usually calls for a Wisdom (Perception) check
- the character must search in the object's vicinity
- searching in the wrong place does not reveal the object, regardless of the roll

Carrying objects:

- ordinary gear and treasure can usually be carried without detailed tracking
- the GM may invoke carrying-capacity rules when the load is unusually heavy or numerous

Breaking objects:

- as an action, a creature can automatically break a fragile nonmagical object such as glass or paper
- destroying something more resilient may use the object-breaking rules from the glossary

Marching order:

- adventurers should establish a travel order while moving
- marching order helps determine trap exposure, detection, and who is nearest when combat begins
- it can be changed outside combat
