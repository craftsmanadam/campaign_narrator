# Tools

Tools are used for specialized ability checks, crafting, or both.

## Tool Entries

Each tool description provides:

- an associated ability
- what can be done with the `Utilize` action
- what can be crafted
- any variants that require separate proficiency

## Tool Proficiency

If you have proficiency with a tool, add your Proficiency Bonus to any ability check you make that uses the tool. If you also have proficiency in a skill used with that check, you have Advantage on the check.

## Important First-Cut Tool Families

### Artisan's Tools

Artisan's tools are used for crafting and trade work. Each requires separate proficiency.

Examples:

- `Calligrapher's Supplies`: Dexterity; can write text that resists forgery; can craft Ink and Spell Scrolls
- `Smith's Tools`: Strength; can pry doors or containers; can craft many weapons, medium armor except Hide, heavy armor, and metal adventuring items
- `Herbalism Kit`: Intelligence; can identify plants; can craft Antitoxin, Candle, Healer's Kit, and Potion of Healing

### Other Notable Tools

- `Thieves' Tools`: Dexterity; pick a lock (`DC 15`) or disarm a trap (`DC 15`)
- `Gaming Set`: Wisdom; detect cheating (`DC 10`) or win a game (`DC 20`)
- `Musical Instrument`: Charisma; play a known tune (`DC 10`) or improvise a song (`DC 15`)
- `Disguise Kit`: Charisma
- `Forgery Kit`: Dexterity

## Crafting Link

The tools chapter links directly to the crafting rules later in `Equipment`, including nonmagical item crafting, brewing Potions of Healing, and scribing Spell Scrolls.
