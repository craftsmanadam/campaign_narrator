# Multiclassing

Multiclassing allows a character to gain levels in multiple classes instead of always taking the next level in the current class.

## Prerequisites

To qualify for a new class, a character must have a score of at least `13` in the primary ability of the new class and in the primary ability of each current class.

## Experience Points and Proficiency Bonus

- XP cost to gain a level is based on total character level, not class level.
- Proficiency Bonus is based on total character level, not level in a particular class.

## Hit Points and Hit Dice

- When multiclassing into a new class, use that class's post-level-1 hit point rule, not its level-1 starting hit points.
- Gain the level-1 hit points for a class only when total character level is `1`.
- Add together the Hit Dice granted by all classes.
- If multiple classes use the same die type, those Hit Dice can be pooled together.
- If classes use different die types, track those Hit Dice separately.

## Proficiencies

When a character gains the first level in a class other than the initial class, only some of that class's starting proficiencies are gained. The exact multiclass proficiency grants are defined in the class descriptions.

## Class Features

When a new level is gained in a class, the features for that class level are gained normally. Some features have additional multiclassing rules in the class descriptions.

## Armor Class and Extra Attack

- If a character has multiple ways to calculate Armor Class, only one AC calculation can apply at a time.
- If Extra Attack is gained from more than one class, the features do not stack.
- A feature such as the Warlock's `Thirsting Blade` also does not grant extra attacks on top of another Extra Attack feature unless a rule specifically says otherwise.

## Multiclass Spellcasting

If a character has the `Spellcasting` feature from more than one class:

- determine prepared spells separately for each class,
- use the spellcasting ability of the class associated with the prepared spell,
- scale cantrips by total character level unless the spell says otherwise.

To determine multiclass spell slots, add together:

- all levels in Bard, Cleric, Druid, Sorcerer, and Wizard,
- half of Paladin and Ranger levels, rounded up.

Then use the Multiclass Spellcaster table for the resulting total. Those spell slots can be used to cast eligible spells prepared from any class with the `Spellcasting` feature.

If a character has both `Pact Magic` and `Spellcasting`:

- Pact Magic slots can cast prepared spells from classes with `Spellcasting`,
- Spellcasting slots can cast prepared Warlock spells.
