# Level Advancement

Characters gain Experience Points while adventuring. When a character's total XP reaches or exceeds a value on the Character Advancement table, the character reaches that level.

## Character Advancement Table

- Level 1: `0 XP`, Proficiency Bonus `+2`
- Level 2: `300 XP`, Proficiency Bonus `+2`
- Level 3: `900 XP`, Proficiency Bonus `+2`
- Level 4: `2,700 XP`, Proficiency Bonus `+2`
- Level 5: `6,500 XP`, Proficiency Bonus `+3`
- Level 6: `14,000 XP`, Proficiency Bonus `+3`
- Level 7: `23,000 XP`, Proficiency Bonus `+3`
- Level 8: `34,000 XP`, Proficiency Bonus `+3`
- Level 9: `48,000 XP`, Proficiency Bonus `+4`
- Level 10: `64,000 XP`, Proficiency Bonus `+4`
- Level 11: `85,000 XP`, Proficiency Bonus `+4`
- Level 12: `100,000 XP`, Proficiency Bonus `+4`
- Level 13: `120,000 XP`, Proficiency Bonus `+5`
- Level 14: `140,000 XP`, Proficiency Bonus `+5`
- Level 15: `165,000 XP`, Proficiency Bonus `+5`
- Level 16: `195,000 XP`, Proficiency Bonus `+5`
- Level 17: `225,000 XP`, Proficiency Bonus `+6`
- Level 18: `265,000 XP`, Proficiency Bonus `+6`
- Level 19: `305,000 XP`, Proficiency Bonus `+6`
- Level 20: `355,000 XP`, Proficiency Bonus `+6`

## Gaining a Level

When a character gains a level, follow these steps in order:

1. Choose a class. Most characters advance in their current class, but a level can instead be taken in another class if the multiclassing prerequisites are met.
2. Adjust hit points and Hit Dice. Gain one additional Hit Die. Either roll that die and add Constitution modifier, or take the fixed class value. The increase is a minimum of `1`.
3. Record new class features. Add the features granted by the class at the new class level and make any choices required by those features.
4. Adjust Proficiency Bonus. If total character level reaches a new Proficiency Bonus threshold, update every derived value that includes Proficiency Bonus.
5. Adjust ability modifiers. If a feat increases an ability score to a new even number, the ability modifier changes. Update every sheet value that uses that modifier.

## Fixed Hit Points by Class

If the table's fixed value is used instead of rolling Hit Dice, add:

- Barbarian: `7 + Constitution modifier`
- Fighter, Paladin, or Ranger: `6 + Constitution modifier`
- Bard, Cleric, Druid, Monk, Rogue, or Warlock: `5 + Constitution modifier`
- Sorcerer or Wizard: `4 + Constitution modifier`

If Constitution modifier increases by `1`, Hit Point maximum increases by `1` for each character level already attained.

## Tiers of Play

The SRD describes four tiers of play:

- Tier 1: levels `1-4`
- Tier 2: levels `5-10`
- Tier 3: levels `11-16`
- Tier 4: levels `17-20`

These tiers describe the changing tone and scale of the campaign. They do not add separate rules by themselves.
