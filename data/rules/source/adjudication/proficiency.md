# Proficiency

Every creature has a Proficiency Bonus that reflects training. Characters gain a larger Proficiency Bonus as they level, and monsters derive it from Challenge Rating.

Proficiency Bonus progression:

- level or CR up to `4`: `+2`
- `5-8`: `+3`
- `9-12`: `+4`
- `13-16`: `+5`
- `17-20`: `+6`
- `21-24`: `+7`
- `25-28`: `+8`
- `29-30`: `+9`

Apply Proficiency Bonus to a D20 Test only when the creature is proficient in the relevant thing being used, such as:

- a skill
- a saving throw
- a weapon
- a tool
- a spell attack

The Proficiency Bonus does not stack. It can never be added to the same roll or number more than once, even if multiple proficiencies appear relevant.

If a rule multiplies or divides Proficiency Bonus, it can be multiplied only once and divided only once for a given use.

Skill proficiency:

- add Proficiency Bonus to an ability check when a relevant skill applies and the creature is proficient
- the GM decides whether the skill is relevant

Saving throw proficiency:

- add Proficiency Bonus to saves using the proficient ability

Equipment proficiency:

- weapon proficiency adds Proficiency Bonus to attack rolls with that weapon
- tool proficiency adds Proficiency Bonus to ability checks using that tool
- if both a relevant skill proficiency and tool proficiency apply to the same tool-based ability check, the creature has Advantage on the check
