# D20 Tests

When the outcome of an action is uncertain, the game uses a D20 Test. There are three kinds:

- ability checks
- saving throws
- attack rolls

Each D20 Test follows this process:

1. Roll `1d20`.
2. If the roll has Advantage or Disadvantage, roll two d20s and use only one result:
   - higher with Advantage
   - lower with Disadvantage
3. Add modifiers:
   - the relevant ability modifier
   - Proficiency Bonus if relevant
   - circumstantial bonuses or penalties from features, spells, or other rules
4. Compare the total to a target number.
   - ability checks and saving throws use a Difficulty Class (`DC`)
   - attack rolls use Armor Class (`AC`)
5. The D20 Test succeeds if the total equals or exceeds the target number; otherwise it fails.
